## How to use it? 

### ⚡️ Live

 ### ✨ Quick start

1.  **Clone this repository.**

    ```sh
    git clone https://github.com/javimostoles/vue-github-users.git
    ```

2.  **Start developing.**

    Navigate into your new site’s directory and start it up.

    ```sh
    cd vue-github-users/
    npm install
    npm run start
    ```

3.  **Running!**

    Your site is now running at `http://localhost:8080` 🚀
